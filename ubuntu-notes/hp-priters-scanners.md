# Install HP Printer / Scanner 
Setup network printer / scanner on Ubuntu.

### Install
```
sudo apt update
sudo apt upgrade
sudo apt install hplip hplip-gui
```

### Setup
```
hp-setup
```

### Use
```
hp-toolbox
```
